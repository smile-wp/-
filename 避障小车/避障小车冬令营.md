## 3D建模课程详细设计（Day 4）

### 教学目标：

1. 理解3D建模基本概念
2. 掌握Fusion360基本操作
3. 能够设计简单的功能外壳

### 教学步骤：

**第一部分：理论讲解（30分钟）**

1. 3D打印原理演示
2. 展示不同3D打印作品（包括小车外壳）
3. 介绍Fusion360平台

**第二部分：软件操作（30分钟）**

1. 基本操作练习：
   - 绘制基本形状
   - 移动、旋转、缩放
   - 布尔运算
   - 挖孔操作（制作安装孔）

**第三部分：设计实践（60分钟）**

1. 提供底盘模板（老师预先设计好）
2. 学生设计任务：
   - 基础任务：设计带孔位的简单外壳
   - 进阶任务：加入个性化元素（火箭尾翼、动物造型等）
3. 设计检查要点：
   - 尺寸适配底盘
   - 预留安装孔位
   - 避免悬空结构（需要支撑）

### 教师准备工作：

- 提前设计2-3种基础外壳模板
- 准备3D打印机的打印平台（调平、上料）
- 安排好打印顺序（按设计复杂度）

## 代码教学调整建议

### 代码注释增加教学提示：

arduino

```
// 教学提示：这个距离参数控制小车的"视力"
// 数值越小，小车"视力"越差，容易撞到
// 数值越大，小车"视力"越好，但可能过于敏感
#define OBSTACLE_DISTANCE 20

// 教学提示：这个速度参数控制小车的"走路速度"
// 数值范围是0-255，数字越大速度越快
// 但太快可能转弯不灵活哦！
#define MAX_SPEED 250
```

## 风险评估与应对

### 3D打印相关风险：

1. **高温风险**：打印头温度200°C以上
   - 应对：设置安全隔离区，学生不得靠近正在打印的机器
2. **长时间打印**：可能无法当天完成
   - 应对：简化设计要求，安排多台打印机并行打印
3. **打印失败**：可能发生堵头、翘边等问题
   - 应对：准备备用外壳，老师提前打印一些通用款

### 课程调整预案：

- **如果3D打印机故障**：改为纯激光切割设计
- **如果网络问题**：使用离线版建模软件
- **如果时间不足**：将3D建模和激光切割合并为一节课

## 扩展建议（可选模块）

### 3D建模进阶内容（针对高年级）：

1. **参数化设计**：使用Fusion 360设计可调整的外壳
2. **功能结构设计**：设计可活动的部件（如可开合舱门）
3. **多部件装配**：设计由多个零件组装的外壳

### 编程扩展功能：

1. **声音反馈**：增加蜂鸣器，不同情况发出不同声音
2. **灯光效果**：增加LED灯带，用灯光表示小车状态
3. **遥控模式**：增加蓝牙模块，可切换自动/手动模式
