# 霍尔系统

## 一、霍尔系统控制舵机和led灯

## 1.完整代码

```
#include <Servo.h>

// 引脚定义
const int HALL_SENSOR_SERVO_PIN = A0;  // 控制舵机的霍尔传感器模拟引脚
const int HALL_SENSOR_LED_PIN = A1;    // 控制LED的霍尔传感器模拟引脚
const int SERVO_PIN = 9;               // 舵机控制引脚
const int LED_PIN = 13;                // LED控制引脚

// 舵机控制参数
const int STOP_PULSE = 1500;           // 停止脉冲 (1500μs)
const int ROTATE_PULSE = 1700;         // 顺时针旋转脉冲 (1700μs)

Servo rotationServo;                   // 舵机对象

// 磁场检测参数
const int MAGNET_THRESHOLD_SERVO = 500; // 舵机控制的磁场触发阈值
const int MAGNET_THRESHOLD_LED = 500;   // LED控制的磁场触发阈值
const int HYSTERESIS = 50;             // 迟滞值防止抖动
bool isRotating = false;               // 当前舵机旋转状态
bool isLedOn = false;                  // 当前LED状态

void setup() {
  // 初始化串口通信
  Serial.begin(9600);
  Serial.println("双霍尔传感器控制系统启动...");
  
  // 初始化舵机
  rotationServo.attach(SERVO_PIN);
  rotationServo.writeMicroseconds(STOP_PULSE);
  
  // 初始化LED引脚
  pinMode(LED_PIN, OUTPUT);
  digitalWrite(LED_PIN, LOW);
  
  Serial.println("系统初始化完成，等待磁场...");
  Serial.println("--------------------------------");
  Serial.println("舵机传感器 | LED传感器 | 舵机状态 | LED状态");
  Serial.println("--------------------------------");
}

void loop() {
  // 读取两个霍尔传感器值 (0-1023)
  int hallServoValue = analogRead(HALL_SENSOR_SERVO_PIN);
  int hallLedValue = analogRead(HALL_SENSOR_LED_PIN);
  
  // 串口输出传感器值
  Serial.print(hallServoValue);
  Serial.print(" | ");
  Serial.print(hallLedValue);
  Serial.print(" | ");
  
  // 控制舵机的霍尔传感器检测
  if (!isRotating && hallServoValue > (MAGNET_THRESHOLD_SERVO + HYSTERESIS)) {
    startRotation();
    isRotating = true;
  }
  else if (isRotating && hallServoValue < (MAGNET_THRESHOLD_SERVO - HYSTERESIS)) {
    stopRotation();
    isRotating = false;
  }
  
  // 控制LED的霍尔传感器检测
  if (!isLedOn && hallLedValue > (MAGNET_THRESHOLD_LED + HYSTERESIS)) {
    digitalWrite(LED_PIN, HIGH);
    isLedOn = true;
    Serial.print("旋转中 | LED开");
  }
  else if (isLedOn && hallLedValue < (MAGNET_THRESHOLD_LED - HYSTERESIS)) {
    digitalWrite(LED_PIN, LOW);
    isLedOn = false;
    Serial.print("停止中 | LED关");
  }
  
  Serial.println(); // 换行
  
  // 短延时保持系统稳定
  delay(100);
}

// 开始旋转函数
void startRotation() {
  rotationServo.writeMicroseconds(STOP_PULSE);
  Serial.print("旋转中 | ");
}

// 停止旋转函数
void stopRotation() {
  rotationServo.writeMicroseconds(ROTATE_PULSE);
  Serial.print("停止中 | ");
}
```



## 2.电路连接图说明

text

```
Arduino Uno
│
├── A0 ──── 霍尔传感器1 (控制舵机) ─── GND
│                │
│                +5V
│
├── A1 ──── 霍尔传感器2 (控制LED) ─── GND
│                │
│                +5V
│
├── 9 ───── 舵机信号线
│      ├── 舵机VCC ─── +5V
│      └── 舵机GND ─── GND
│
└── 13 ──── LED正极 ──┬─ 220Ω电阻 ── GND
                      └─ (可选)
```

### 连接说明：

1. 两个霍尔传感器分别连接到A0和A1模拟输入引脚
2. 每个霍尔传感器需要连接VCC(5V)和GND
3. 舵机连接到数字引脚9，并需要VCC和GND
4. LED连接到数字引脚13，建议串联220Ω限流电阻

### 工作原理：

- 当磁铁靠近控制舵机的霍尔传感器时，舵机开始旋转
- 当磁铁远离时，舵机停止
- 当磁铁靠近控制LED的霍尔传感器时，LED点亮
- 当磁铁远离时，LED熄灭

### 注意事项：

1. 根据实际霍尔传感器的输出值调整阈值(MAGNET_THRESHOLD_SERVO和MAGNET_THRESHOLD_LED)
2. 如果需要改变旋转方向，可以修改ROTATE_PULSE的值(1300为逆时针，1700为顺时针)
3. 确保电源能提供足够的电流，特别是当舵机和LED同时工作时

# 三、加湿器

## 1.基础硬件配置

1. **核心组件**：
   - Arduino Uno/Nano开发板
   - 5V继电器模块（推荐带光耦隔离）
   - 加湿器（AC 220V或DC 24V等）
2. **控制输入**：
   - 压力传感器（FSR402）
   - 10kΩ电阻（用于分压电路）
   - 可选：DHT11温湿度传感器
3. **保护元件**：
   - 1N4007二极管（继电器线圈保护）
   - 自恢复保险丝（推荐2A）

## 4.三种实用控制模式

### 1. 定时间歇模式

arduino

```
const int WORK_TIME = 30000;  // 工作30秒
const int REST_TIME = 10000;  // 休息10秒

void loop() {
  if(analogRead(PRESSURE_PIN) > THRESHOLD) {
    unsigned long currentTime = millis();
    
    // 计算当前周期内的时间余数
    unsigned long cycleTime = currentTime % (WORK_TIME + REST_TIME);
    
    if(cycleTime < WORK_TIME) {
      digitalWrite(RELAY_PIN, HIGH);
    } else {
      digitalWrite(RELAY_PIN, LOW);
    }
  } else {
    digitalWrite(RELAY_PIN, LOW);
  }
}
```

### 2. 压力梯度控制

arduino

```
void loop() {
  int pressure = analogRead(PRESSURE_PIN);
  
  if(pressure > 600) {         // 高压：持续工作
    digitalWrite(RELAY_PIN, HIGH);
  } 
  else if(pressure > 400) {    // 中压：间歇工作
    digitalWrite(RELAY_PIN, (millis() % 20000) < 15000);
  }
  else if(pressure > 200) {    // 低压：短时工作
    digitalWrite(RELAY_PIN, (millis() % 10000) < 3000);
  }
  else {                       // 无压力：关闭
    digitalWrite(RELAY_PIN, LOW);
  }
}
```

### 3. 智能湿度维持

arduino

```
const int TARGET_HUMIDITY = 60;  // 目标湿度(%)

void loop() {
  float humidity = dht.readHumidity();

}
```

## 5.安全注意事项

1. **高压隔离**：

   - 使用质量合格的继电器模块
   - 保持高压线路与低压线路至少5mm间距

2. **防潮措施**：

   - 电路板喷涂三防漆
   - 使用防水外壳（IP65等级以上）

3. **电气保护**：

   arduino

   ```
   // 在代码中添加电流检测保护
   const int CURRENT_PIN = A1;
   const int MAX_CURRENT = 500;  // 500mA
   
   void checkCurrent() {
     int current = analogRead(CURRENT_PIN) * 0.488;  // 转换为mA
     if(current > MAX_CURRENT) {
       digitalWrite(RELAY_PIN, LOW);
       Serial.println("Overcurrent protection!");
       delay(60000);  // 锁定1分钟
     }
   }
   ```

4. **机械保护**：

   - 安装水流检测传感器
   - 添加倾斜开关防止倾倒

## 6.常见问题解决

**问题1：继电器频繁开关**

- 解决方案：添加迟滞控制

  arduino

  ```
  const int HYSTERESIS = 50;
  bool relayState = false;
  
  void loop() {
    int pressure = analogRead(PRESSURE_PIN);
    
    if(pressure > THRESHOLD + HYSTERESIS) {
      relayState = true;
    } 
    else if(pressure < THRESHOLD - HYSTERESIS) {
      relayState = false;
    }
    
    digitalWrite(RELAY_PIN, relayState);
  }
  ```

**问题2：压力传感器读数不稳**

- 解决方案：软件滤波

  arduino

  ```
  int stablePressureRead() {
    const int SAMPLES = 10;
    int sum = 0;
    
    for(int i=0; i<SAMPLES; i++) {
      sum += analogRead(PRESSURE_PIN);
      delay(5);
    }
    return sum / SAMPLES;
  }
  ```

**问题3：加湿器启动电流大**

- 解决方案：软启动控制

  arduino

  ```
  void softStart() {
    for(int i=0; i<3; i++) {
      digitalWrite(RELAY_PIN, HIGH);
      delay(100);
      digitalWrite(RELAY_PIN, LOW);
      delay(500);
    }
    digitalWrite(RELAY_PIN, HIGH);
  }
  ```

这个方案可根据实际需求灵活调整，建议开发时遵循以下流程：

1. 先用LED测试控制逻辑
2. 接低压负载测试继电器
3. 最后接入实际加湿器
4. 逐步增加保护功能







# 四、舵机sg90旋转台

## 1.所需组件：

1. 舵机（如SG90）
2. 霍尔传感器（或干簧管）和磁铁
3. 微控制器（如Arduino）

## 2.工作原理：

- 当磁铁靠近霍尔传感器时，传感器输出低电平（或高电平，取决于类型和接线）；远离时输出相反电平。
- 每次检测到磁铁靠近时，舵机在0度和180度之间切换状态（即每次靠近时旋转到相反位置）。
- 磁铁远离时，舵机保持当前位置。

## 3.连接方式（以Arduino为例）：

- 舵机信号线 -> Arduino数字引脚9
- 霍尔传感器VCC -> Arduino 5V
- 霍尔传感器GND -> Arduino GND
- 霍尔传感器OUT -> Arduino数字引脚2（使用中断引脚以便响应及时）

## 4.代码示例：

cpp

```
#include <Servo.h>

Servo myservo;  // 创建舵机对象

const int hallSensorPin = 2;  // 霍尔传感器连接引脚
int servoAngle = 0;           // 当前舵机角度（0或180）
bool lastHallState = HIGH;    // 上一次霍尔传感器状态（假设初始为高，即无磁铁）
bool currentHallState;        // 当前霍尔传感器状态

void setup() {
  myservo.attach(9);          // 舵机连接引脚9
  pinMode(hallSensorPin, INPUT_PULLUP); // 启用内部上拉电阻，这样无磁铁时输入为高，有磁铁时低
  myservo.write(servoAngle);   // 初始化舵机角度为0
}

void loop() {
  currentHallState = digitalRead(hallSensorPin);

  // 检测磁铁靠近（下降沿：从高到低）
  if (lastHallState == HIGH && currentHallState == LOW) {
    // 切换舵机角度
    if (servoAngle == 0) {
      servoAngle = 180;
    } else {
      servoAngle = 0;
    }
    myservo.write(servoAngle); // 转动舵机
    delay(500);                // 防抖延时，避免多次触发
  }

  lastHallState = currentHallState; // 更新状态
}
```

说明：

1. 使用内部上拉电阻，因此无磁铁时传感器输出高电平，有磁铁时输出低电平。
2. 检测下降沿（从高到低）表示磁铁靠近。
3. 每次磁铁靠近时，舵机角度在0和180之间切换。
4. `delay(500)`用于防抖，避免磁铁抖动导致多次触发。您可能需要根据实际情况调整延时时间。

### 改进建议：

- 如果需要更可靠的触发，可以使用中断（将霍尔传感器连接到中断引脚，如Arduino Uno的引脚2或3）。
- 使用中断的代码示例（更及时响应）：

cpp

```
#include <Servo.h>

Servo myservo;
volatile bool flag = false;
int servoAngle = 0;

void setup() {
  myservo.attach(9);
  pinMode(2, INPUT_PULLUP);
  attachInterrupt(digitalPinToInterrupt(2), magnetDetected, FALLING);
  myservo.write(servoAngle);
}

void loop() {
  if (flag) {
    // 在中断中只设置标志位，主循环中处理转动，避免在中断中执行耗时操作
    if (servoAngle == 0) {
      servoAngle = 180;
    } else {
      servoAngle = 0;
    }
    myservo.write(servoAngle);
    delay(500); // 防抖，同时避免重复触发
    flag = false;
  }
}

void magnetDetected() {
  flag = true;
}
```

### 注意事项：

- 根据您的霍尔传感器类型（常开或常闭），可能需要调整触发逻辑（例如使用RISING代替FALLING）。
- 如果磁铁保持靠近状态，舵机只会触发一次，直到磁铁远离并再次靠近才会下次触发。

请根据您的实际硬件调整代码。



# 五、点亮led

## 项目概述

**目标：** 使用一个霍尔传感器来检测磁场，当有磁铁靠近时，LED灯亮起；当磁铁远离时，LED灯熄灭。

------

## 1、所需材料

1. **微控制器 (MCU)：** 最常用的是 **Arduino Uno**（或其他Arduino板、ESP32等）。
2. **霍尔传感器：**
   - **数字开关型（如 A3144, OH44E, US1881）：** 最常用，输出简单的高低电平。磁铁南极（或北极，取决于型号）靠近时输出低电平，远离时输出高电平。
   - **线性模拟型（如 SS49E）：** 输出随磁场强度变化的模拟电压值，需要更复杂的程序判断，但可测量磁场强弱。
3. **LED灯:** 一个普通的发光二极管。
4. **电阻：**
   - 1个 **220Ω** 的电阻（用于LED，限流保护）。
   - 1个 **10kΩ** 的电阻（如果使用数字传感器，用于上拉，但Arduino内部上拉也可替代）。
5. **磁铁：** 一块普通的钕磁铁（强磁铁效果更明显）。
6. **面包板和杜邦线**：用于连接电路。

**本项目我们以更常见的数字开关型霍尔传感器（A3144）为例。**

------

## 2、工作原理（数字传感器）

数字霍尔传感器内部有一个晶体管开关。当没有磁场时，传感器输出高电平；当正确极性的磁铁（通常是南极）靠近传感器有标记的一面时，内部的晶体管导通，输出引脚被拉低到低电平。

我们可以通过微控制器（如Arduino）检测这个电平变化，从而控制LED的亮灭。

------

## 3、电路连接（以Arduino为例）

#### 连接步骤：

1. **霍尔传感器：**
   - 将传感器的 **VCC** 引脚连接到 Arduino的 **5V** 引脚。
   - 将传感器的 **GND** 引脚连接到 Arduino的 **GND** 引脚。
   - 将传感器的 **OUT** (信号) 引脚连接到 Arduino的 **数字引脚 2** (或其他任何数字引脚)。
2. **LED灯：**
   - 将LED的**长脚（正极，阳极）** 通过一个 **220Ω** 的限流电阻连接到 Arduino的 **数字引脚 13**。
   - 将LED的**短脚（负极，阴极）** 直接连接到 Arduino的 **GND** 引脚。

**电路示意图：**

text

```
Arduino Uno：
    5V ----> Hall Sensor VCC
    GND ----> Hall Sensor GND & LED Cathode (-)
    Pin 2 ----> Hall Sensor OUT
    Pin 13 ----> 220Ω Resistor ----> LED Anode (+)
```

------

## 4、Arduino 代码

这里提供两个示例代码：一个基础版本，一个使用内部上拉电阻的稳定版本。

#### 示例代码 1：基础代码

cpp

```
// 定义引脚
const int hallSensorPin = 2; // 霍尔传感器连接到数字引脚2
const int ledPin = 13;       // LED连接到数字引脚13

void setup() {
  // 初始化串口通信（用于调试，可选）
  Serial.begin(9600);
  
  // 设置LED引脚为输出模式
  pinMode(ledPin, OUTPUT);
  
  // 设置传感器引脚为输入模式
  pinMode(hallSensorPin, INPUT);
}

void loop() {
  // 读取霍尔传感器引脚的值
  int sensorState = digitalRead(hallSensorPin);

  // 打印传感器状态到串口监视器（可选）
  Serial.println(sensorState);

  // 判断：如果传感器输出低电平（磁铁靠近）
  if (sensorState == LOW) {
    digitalWrite(ledPin, HIGH); // 点亮LED
    Serial.println("Magnet detected! LED ON");
  } else {
    digitalWrite(ledPin, LOW);  // 熄灭LED
    Serial.println("No magnet. LED OFF");
  }

  delay(500); // 短暂延迟，使输出更稳定，可调整或删除
}
```

#### 示例代码 2：使用内部上拉电阻（推荐，更稳定）

Arduino的引脚可以启用内部上拉电阻，这样就不需要外接一个物理的10kΩ上拉电阻了，电路更简洁。

cpp

```
// 定义引脚
const int hallSensorPin = 2; // 霍尔传感器连接到数字引脚2
const int ledPin = 13;       // LED连接到数字引脚13

void setup() {
  Serial.begin(9600);
  
  pinMode(ledPin, OUTPUT);
  
  // 设置传感器引脚为输入模式，并启用内部上拉电阻
  pinMode(hallSensorPin, INPUT_PULLUP);
  // 启用后，默认情况下sensorState读到的就是HIGH
}

void loop() {
  // 读取霍尔传感器引脚的值
  int sensorState = digitalRead(hallSensorPin);

  // 由于启用了内部上拉，常态下为HIGH，磁铁靠近时传感器将其拉低为LOW
  if (sensorState == LOW) { 
    digitalWrite(ledPin, HIGH); // 磁铁靠近，灯亮
    Serial.println("Magnet detected! LED ON");
  } else {
    digitalWrite(ledPin, LOW);  // 磁铁远离，灯灭
    Serial.println("No magnet. LED OFF");
  }

  delay(100); // 更快的响应
}
```

## 5、测试与调试

1. **上传代码：** 将代码上传到你的Arduino板。
2. **打开串口监视器：** 在Arduino IDE中打开串口监视器（右上角的放大镜图标），设置波特率为9600。你会看到不断打印的传感器状态。
3. **用磁铁测试：** 拿磁铁慢慢靠近霍尔传感器有字的一面。你会发现当磁铁靠近到一定距离时，LED灯突然亮起，同时串口监视器会显示 `"Magnet detected! LED ON"`。拿开磁铁，LED灯熄灭。

**常见问题：**

- **LED不亮：** 检查线路连接是否正确，LED正负极是否接反。
- **没磁铁也常亮或常灭：** 检查代码中`INPUT_PULLUP`设置是否正确，磁铁的极性是否正确（尝试用另一面靠近，或者换一个磁铁）。串口监视器的输出可以帮助你判断传感器是否正常工作。

### 进阶应用

- **控制大功率LED/灯带：** 可以使用继电器或MOSFET管，用Arduino的输出来控制更大功率的灯具。
- **速度测量：** 将霍尔传感器和磁铁分别固定在轮子和车架上，可以测量轮子的转速（每转一圈产生一个脉冲）。
- **位置检测：** 用于检测门、窗是否关闭（在门框上装传感器，在门上装磁铁）。
- **使用线性霍尔传感器：** 可以制作不仅检测有无，还能检测磁场强弱的项目，例如“磁力计”。
