# 月卡课大纲

## 课程体系总览

### 第1部分：基础入门系列（2课时）

- **基础课1：认识Arduino Uno板和IDE软件**
- **基础课2：电路入门与LED控制**

### 第2部分：核心编程系列（3课时）

- **LED灯和灯带编程**（5种灯光效果）
- **舵机控制课程**（180度和360度舵机）
- **MiniPlayer MP3模块**（音乐播放与控制）

### 第3部分：传感器与应用系列（1课时）

- **霍尔传感器及应用**（综合项目）

------

## 基础课1：认识Arduino Uno板和IDE软件（90分钟）

### 🎯 学习目标

1. 认识Arduino Uno板的主要部件
2. 学会安装和配置Arduino IDE软件
3. 完成第一个程序上传并理解程序结构
4. 学会使用串口监视器调试程序

### 📚 课程内容

- **课程导入**（10分钟）
- **理论知识讲解**（20分钟）
  - Arduino是什么
  - Arduino Uno板详细讲解
  - Arduino IDE安装演示
- **动手实践**（40分钟）
  - 环境搭建
  - 第一个程序Blink
  - 串口监视器使用
- **知识总结与扩展**（15分钟）
  - 知识总结
  - 扩展活动：创意闪烁（SOS求救信号）
  - 家庭作业

------

## 基础课2：电路入门与LED控制（90分钟）

### 🎯 学习目标

1. 认识基本电子元件和面包板结构
2. 学会安全连接简单电路
3. 掌握LED的控制原理
4. 学会使用电阻保护LED

### 📚 课程内容

- **课程导入**（10分钟）
- **理论知识讲解**（20分钟）
  - 电子元件基础
  - 面包板结构详解
  - 电路基础概念
- **动手实践**（40分钟）
  - 安全须知
  - 搭建第一个外部LED电路
  - 多LED控制（交通信号灯模拟）
  - 故障排除练习
- **知识总结与扩展**（15分钟）
  - 知识总结
  - 扩展活动：创意灯光秀（生日蜡烛效果）
  - 家庭作业

## LED灯和灯带编程（90分钟）

### 🎯 学习目标

1. 理解PWM（脉宽调制）原理
2. 掌握单LED呼吸灯实现
3. 学会控制RGB LED显示不同颜色
4. 掌握WS2812B灯带的基本控制
5. 创建至少5种灯光效果

### 📚 课程内容

- **课程导入**（10分钟）
- **理论知识讲解**（20分钟）
  - PWM脉宽调制原理
  - RGB颜色模型
  - WS2812B智能灯带
- **动手实践**（40分钟）
  - 实践1：单LED呼吸灯
  - 实践2：RGB LED调色板
  - 实践3：WS2812B灯带控制（5种效果）
    - 彩虹流动
    - 呼吸效果
    - 流水灯
    - 颜色擦除
    - 烟花效果
- **知识总结与扩展**（15分钟）
  - 知识总结
  - 扩展活动：音乐可视化灯带
  - 家庭作业

------

  - 实践3：按键控制播放器
- **知识总结与扩展**（15分钟）
  - 知识总结
  - 扩展活动：智能音乐盒（根据环境光自动播放）
  - 家庭作业

- **动手实践**（40分钟）

  - 实践3：霍尔传感器控制舵机（磁控门锁）
- **知识总结与扩展**（15分钟）
  - 知识总结

## 课程特色亮点

### 🌟 分层次教学

- **基础层**：硬件认知、电路搭建
- **应用层**：传感器使用、模块控制
- **创新层**：综合项目、创意扩展

### 🌟 实用技能培养

- 电路设计能力
- 编程逻辑思维
- 故障排除技巧
- 项目规划能力

### 🌟 安全规范教育

- 电子实验安全准则
- 正确使用工具方法
- 故障预防与处理

### 🌟 创意激发引导

- 每节课扩展活动
- 开放式家庭作业
- 作品展示与分享

------

**课时总计**：6节课 × 90分钟 = 9小时

**适合年龄**：7-17岁（根据年龄调整深度和难度）

**先修要求**：无（零基础入门）

**课程产出**：每个学生完成6个实践项目 + 1个综合项目
