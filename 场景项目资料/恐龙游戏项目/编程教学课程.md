## 第一部分：引入仿真概念（1-2课时）

### 1.1 什么是仿真？

**教学要点：**

- 仿真就像在电脑上"假装"搭建电路
- 可以在不买真实零件的情况下测试想法
- 如果仿真出错，不会损坏真实零件，很安全

**简单比喻：**
"就像在玩Minecraft时搭建房子，先试试看好不好看，再在真实世界里建造"

### 1.2 Proteus仿真软件介绍

**基本概念：**

- Proteus是我们的"虚拟实验室"
- 可以拖放电子元件就像搭积木
- 可以看到电流如何流动

**操作练习：**

1. 打开Proteus软件
2. 认识工作区：元件库、绘图区、仿真控制
3. 学习保存项目（重要！）

### 2.1 Arduino IDE介绍

**界面认识：**

- 代码编辑区（写指令的地方）
- 验证按钮（检查代码对不对）
- 上传按钮（把代码发送给Arduino）

### 2.2 第一个程序：让LED闪烁

```c++
// 这是让LED闪烁的程序
void setup() {
  // setup里面的代码只运行一次
  pinMode(13, OUTPUT); // 告诉Arduino，13号引脚要控制灯
}

void loop() {
  // loop里面的代码会一直重复运行
  digitalWrite(13, HIGH); // 开灯
  delay(1000);            // 等待1秒钟
  digitalWrite(13, LOW);  // 关灯
  delay(1000);            // 等待1秒钟
}
```

**代码解释游戏：**

- `pinMode`：告诉Arduino某个引脚要做什么
- `digitalWrite`：让引脚输出高电平(开)或低电平(关)
- `delay`：等待一段时间

### 2.3 生成HEX文件

**步骤：**

1. 点击"文件" → "首选项"
2. 勾选"编译时显示详细输出"
3. 点击验证（✓）
4. 在输出信息中找到临时hex文件路径

## 第三部分：第一个升降模块教学（2课时）

### 3.1 硬件认识

**KY003霍尔传感器：**

- 像一个小侦探，能发现附近的磁铁
- 有3条腿：VCC(吃饭)、GND(休息)、OUT(说话)

**MG90舵机：**

- 像机器人的关节，可以转动到指定角度
- 有3条线：红色(吃饭)、棕色(休息)、橙色(听命令)

### 3.2 Proteus中搭建电路

**连接步骤：**

1. 从元件库找到：
   - ARDUINO UNO R3
   - KY003 (用开关代替)
   - SERVO-MOTOR
2. 连线：
   - KY003 OUT → Arduino 引脚2
   - 舵机信号线 → Arduino 引脚9
   - 所有VCC接5V，所有GND接GND

### 3.3 编写第一个升降模块代码

```c++
#include <Servo.h> // 请来舵机帮手

Servo myServo1;    // 给第一个舵机起名字
int hallPin1 = 2;  // 霍尔传感器接在2号引脚
int servoPin1 = 9; // 舵机接在9号引脚

void setup() {
  pinMode(hallPin1, INPUT_PULLUP); // 告诉Arduino要听霍尔传感器说话
  myServo1.attach(servoPin1);      // 告诉舵机要听命令
  myServo1.write(0);               // 让舵机回到0度位置
}

void loop() {
  int hallState1 = digitalRead(hallPin1); // 问问霍尔传感器看到了什么
  
  if (hallState1 == LOW) { // 如果看到磁铁(LOW表示看到)
    myServo1.write(120);   // 舵机转到120度
    delay(1000);           // 保持1秒钟
  } else {
    myServo1.write(0);     // 没看到磁铁就回到0度
  }
}
```

## 第四部分：第二个升降模块（1课时）

### 4.1 添加第二个模块

```c++
#include <Servo.h>

Servo myServo1, myServo2; // 两个舵机
int hallPin1 = 2, hallPin2 = 3;
int servoPin1 = 9, servoPin2 = 10;

void setup() {
  pinMode(hallPin1, INPUT_PULLUP);
  pinMode(hallPin2, INPUT_PULLUP);
  myServo1.attach(servoPin1);
  myServo2.attach(servoPin2);
  myServo1.write(0);
  myServo2.write(0);
}

void loop() {
  // 第一个传感器和舵机
  if (digitalRead(hallPin1) == LOW) {
    myServo1.write(120);
  } else {
    myServo1.write(0);
  }
  
  // 第二个传感器和舵机
  if (digitalRead(hallPin2) == LOW) {
    myServo2.write(120);
  } else {
    myServo2.write(0);
  }
  
  delay(100);
}
```

## 第八部分：翼龙翅膀模块（1课时）

```c++
#include <Servo.h>

Servo wingServo;      // 翅膀舵机
int wingHallPin = A0;
int wingServoPin = A1;
bool wingMoving = false; // 记录翅膀是否在动

void setup() {
  // ... 前面的setup代码
  pinMode(wingHallPin, INPUT_PULLUP);
  wingServo.attach(wingServoPin);
  wingServo.write(90); // 360度舵机90表示停止
}

void loop() {
  // ... 前面的loop代码
  
  // 翼龙翅膀控制
  if (digitalRead(wingHallPin) == LOW) {
    delay(50); // 防抖动
    if (digitalRead(wingHallPin) == LOW) {
      wingMoving = !wingMoving; // 切换状态
      
      if (wingMoving) {
        wingServo.write(180); // 开始转动
      } else {
        wingServo.write(90);  // 停止转动
      }
      
      while(digitalRead(wingHallPin) == LOW) {
        delay(10); // 等待传感器释放
      }
    }
  }
}
```

## 第十部分：完整代码整合（2课时）

```c++
#include <Servo.h>

// 定义所有舵机
Servo myServo1, myServo2, jawServo, doorServo, wingServo;

// 定义传感器引脚
int hallPin1 = 2, hallPin2 = 3, jawHallPin = 4;
int doorHallPin = 5, ledHallPin = 12, fireHallPin = 13;
int wingHallPin = A0;

// 定义舵机引脚
int servoPin1 = 9, servoPin2 = 10, jawServoPin = 11;
int doorServoPin = 6, wingServoPin = A1;

// 定义LED引脚
int ledPin = 7, firePin = 8;

// 定义蜂鸣器引脚
int buzzerPin = A2;

// 翅膀状态
bool wingMoving = false;

void setup() {
  // 设置传感器引脚
  pinMode(hallPin1, INPUT_PULLUP);
  pinMode(hallPin2, INPUT_PULLUP);
  pinMode(jawHallPin, INPUT_PULLUP);
  pinMode(doorHallPin, INPUT_PULLUP);
  pinMode(ledHallPin, INPUT_PULLUP);
  pinMode(fireHallPin, INPUT_PULLUP);
  pinMode(wingHallPin, INPUT_PULLUP);
  
  // 连接舵机
  myServo1.attach(servoPin1);
  myServo2.attach(servoPin2);
  jawServo.attach(jawServoPin);
  doorServo.attach(doorServoPin);
  wingServo.attach(wingServoPin);
  
  // 设置LED引脚
  pinMode(ledPin, OUTPUT);
  pinMode(firePin, OUTPUT);
  
  // 设置蜂鸣器引脚
  pinMode(buzzerPin, OUTPUT);
  
  // 初始化位置
  myServo1.write(0);
  myServo2.write(0);
  jawServo.write(0);
  doorServo.write(0);
  wingServo.write(90);
}

void playSound(int type) {
  // 音效播放函数（同上）
}

void loop() {
  // 检查所有传感器并控制对应设备
  
  // 升降模块1
  if (digitalRead(hallPin1) == LOW) {
    myServo1.write(120);
    playSound(1);
  } else {
    myServo1.write(0);
  }
  
  // 升降模块2
  if (digitalRead(hallPin2) == LOW) {
    myServo2.write(120);
    playSound(1);
  } else {
    myServo2.write(0);
  }
  
  // 牙齿咬合
  if (digitalRead(jawHallPin) == LOW) {
    jawServo.write(60);
    playSound(2);
  } else {
    jawServo.write(0);
  }
  
  // 门开关
  if (digitalRead(doorHallPin) == LOW) {
    doorServo.write(90);
    playSound(3);
  } else {
    doorServo.write(0);
  }
  
  // LED灯
  digitalWrite(ledPin, digitalRead(ledHallPin) == LOW ? HIGH : LOW);
  
  // 篝火灯
  digitalWrite(firePin, digitalRead(fireHallPin) == LOW ? HIGH : LOW);
  
  // 翼龙翅膀
  if (digitalRead(wingHallPin) == LOW) {
    delay(50);
    if (digitalRead(wingHallPin) == LOW) {
      wingMoving = !wingMoving;
      if (wingMoving) {
        wingServo.write(180);
        playSound(4);
      } else {
        wingServo.write(90);
      }
      while(digitalRead(wingHallPin) == LOW) {
        delay(10);
      }
    }
  }
  
  delay(100);
}
```

## 教学建议

### 课时安排：

- 总共约14-16课时
- 每课时45分钟

### 教学技巧：

1. **实物展示**：展示真实的Arduino、传感器和舵机
2. **分步教学**：每个模块单独教学，确保掌握后再继续
3. **错误排查**：教会学生如何阅读错误信息
4. **鼓励创新**：在基础功能完成后，鼓励学生添加自己的创意

### 评估方式：

- 完成每个模块的仿真
- 能够解释代码的作用
- 能够排查简单的连接错误
