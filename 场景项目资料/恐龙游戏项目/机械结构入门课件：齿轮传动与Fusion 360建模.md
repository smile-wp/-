# 机械结构入门课件：齿轮传动与Fusion 360建模

## 第一部分：齿轮基础知识（2课时）

### 1.1 什么是齿轮？

**生动比喻：**
"齿轮就像机器世界的'好朋友手拉手'，一个齿轮转动时，会拉着另一个齿轮一起转"

**基本概念：**

- 齿轮：边缘有齿的轮子
- 齿：齿轮上的"小牙齿"，用来互相咬合
- 传动：通过齿轮传递运动和力量

### 1.2 常见齿轮类型

![](D:\交互资料代码\恐龙游戏项目\齿轮.jpg)

#### 直齿轮（最基础）

![](D:\交互资料代码\恐龙游戏项目\直齿轮.png)
**特点：**

- 齿是直的，平行于轴线
- 像两个自行车齿轮并排转动
- **应用：** 钟表、简单玩具

**课堂活动：** 用两个圆形纸板剪出齿状，让学生体验直齿轮传动

#### 锥齿轮（改变方向）

![](D:\交互资料代码\恐龙游戏项目\锥齿轮.png)
**特点：**

- 形状像圆锥
- 可以把转动方向改变90度
- **应用：** 汽车差速器、钻床

**生动例子：** "就像两个人从不同方向握手"

#### 蜗轮蜗杆（减速增力）

![](D:\交互资料代码\恐龙游戏项目\蜗杆齿轮.jpg)
**特点：**

- 蜗杆像螺丝，蜗轮像特殊齿轮
- 传动比大，能大幅减速
- 只能由蜗杆带动蜗轮，反向会自锁
- **应用：** 升降机、卷帘门

**课堂演示：** 用螺丝和齿轮演示自锁原理

#### 齿条齿轮（转动变直线运动）

![](D:\交互资料代码\恐龙游戏项目\齿轮齿条.jpg)
**特点：**

- 齿轮转动，齿条直线移动
- 把旋转运动变成直线运动
- **应用：** 汽车方向盘、推拉门

### 1.3 齿轮传动比

**简单计算：**

text

```
传动比 = 从动齿轮齿数 ÷ 主动齿轮齿数
```



**例子游戏：**

- 大齿轮40齿，小齿轮10齿
- 传动比 = 40 ÷ 10 = 4
- 小齿轮转4圈，大齿轮才转1圈

**力量变化：**

- 速度变慢 → 力量变大
- 速度变快 → 力量变小

### 2.2 创建第一个齿轮（直齿轮）

#### 方法一：使用齿轮生成器

详细步骤

```
1. 点击"设计"工作空间
2. 选择"插入" → "插入MFG零件"
3. 选择"Spur Gear"（直齿轮）
4. 设置参数：
   - 模数：1（齿的大小）
   - 齿数：20
   - 厚度：5mm
5. 点击确定生成齿轮
```



#### 方法二：手动绘制（理解原理）

详细步骤

```
1. 创建草图：
   - 点击"创建草图"
   - 选择XY平面

2. 绘制基圆：
   - 点击"圆"工具
   - 画一个直径20mm的圆

3. 绘制齿形：
   - 使用"线"工具画出单个齿形
   - 齿高约2.25mm
   - 齿厚约1.57mm

4. 阵列齿形：
   - 选择画好的齿形
   - 点击"圆形阵列"
   - 数量：20个
   - 角度：360度

5. 拉伸成3D：
   - 完成草图
   - 点击"拉伸"
   - 厚度：5mm
```



### 2.3 创建第二个配对齿轮

步骤

```
1. 重复上述步骤
2. 改变齿数：40齿
3. 直径：40mm
4. 确保模数相同（都是1）
```



## 第三部分：门开合齿轮机构设计（4课时）

### 3.1 设计思路

**机构组成：**

1. **主动齿轮：** 连接舵机，小齿轮（20齿）
2. **从动齿轮：** 连接门轴，大齿轮（40齿）
3. **门板：** 固定在从动齿轮上
4. **支架：** 支撑所有零件

**传动原理：**

- 舵机转动90度 → 主动齿轮转90度
- 传动比2:1 → 从动齿轮转45度
- 门板随之转动45度（半开状态）

### 3.2 创建齿轮组

#### 步骤1：创建主动齿轮（20齿）

Fusion

```
1. 使用齿轮生成器创建直齿轮
2. 参数：
   - 模数：1
   - 齿数：20
   - 厚度：3mm
   - 轴孔直径：3mm（适配舵机轴）
```



#### 步骤2：创建从动齿轮（40齿）

Fusion

```
1. 创建新齿轮
2. 参数：
   - 模数：1（必须相同！）
   - 齿数：40
   - 厚度：3mm
   - 轴孔直径：2mm（连接门轴）
```



#### 步骤3：检查齿轮配合

重要检查

```
1. 中心距计算：
   - 公式：(齿数1 + 齿数2) × 模数 ÷ 2
   - (20 + 40) × 1 ÷ 2 = 30mm

2. 装配验证：
   - 将两个齿轮中心距离设为30mm
   - 旋转检查齿是否干涉
```



### 3.3 创建门和支架

#### 步骤4：创建门板

建模步骤

```
1. 创建新组件
2. 绘制矩形草图：60mm × 40mm
3. 拉伸厚度：2mm
4. 添加门轴：
   - 在门边中心画直径2mm的圆
   - 拉伸高度10mm
```



#### 步骤5：创建支架

建模步骤

```
1. 创建底板：
   - 矩形80mm × 40mm
   - 厚度3mm


2. 添加支撑柱：
   - 在齿轮中心位置画立柱
   - 高度15mm，直径8mm

3. 添加舵机座：
   - 根据MG90舵机尺寸创建安装座
   - 预留螺丝孔
```



### 3.4 完整装配

#### 步骤6：组装所有零件

装配顺序

```
1. 固定底板
2. 安装支架和舵机座
3. 放置从动齿轮，连接门轴
4. 放置主动齿轮，连接舵机轴
5. 调整齿轮间距至30mm
6. 检查转动是否顺畅
```



#### 步骤7：添加运动约束

Fusion

```
1. 进入"装配"工作空间
2. 选择"关节"工具
3. 为主动齿轮添加旋转关节
4. 为从动齿轮添加旋转关节
5. 添加齿轮接触约束
```



## 第四部分：运动仿真（2课时）

### 4.1 设置运动仿真

步骤

```
1. 切换到"仿真"工作空间
2. 选择"运动仿真"
3. 添加重力（可选）
4. 设置接触集合：
   - 选择两个齿轮的齿面
   - 设置摩擦系数
```



### 4.2 添加驱动和测试

仿真设置

```
1. 添加旋转驱动：
   - 选择主动齿轮
   - 运动类型：速度
   - 速度：30 deg/s（模拟舵机）

2. 运行仿真：
   - 点击"仿真播放"
   - 观察门开合动作
   - 检查是否有干涉
```



### 4.3 优化设计

问题排查

```
1. 齿间干涉：调整中心距
2. 转动不顺畅：检查轴孔配合
3. 门开合角度：调整齿轮齿数比
```



## 第五部分：3D打印准备（2课时）

### 5.1 导出零件

导出步骤

```
1. 分别选择每个零件
2. 右键 → "另存为网格"
3. 格式：STL
4. 分辨率：高
```



### 5.2 3D打印注意事项

打印设置

```
1. 齿轮：
   - 层高：0.1mm（高精度）
   - 填充率：40%
   - 支撑：需要（对于悬垂齿）

2. 支架和门：
   - 层高：0.2mm
   - 填充率：20%
   - 支撑：根据情况
```
